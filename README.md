# Система кредитного скоринга на основе алгоритмов дерева решений

## Описание проекта

Данный проект посвящён разработке системы кредитного скоринга, предназначенной для оценки кредитоспособности заёмщиков на основе методов машинного обучения.  
В качестве основных алгоритмов используются **дерево решений (Decision Tree)** и **случайный лес (Random Forest)**.

Система позволяет:
- вводить параметры заёмщика через веб-интерфейс;
- получать вероятность кредитного риска (класс 1 — дефолт);
- принимать решение об одобрении или отказе в кредите;
- анализировать качество модели и влияние признаков.

Проект реализован в виде веб-приложения с backend на Flask и моделью машинного обучения на scikit-learn.

---

## Цель и задачи проекта

### Цель проекта  
Разработать и реализовать систему кредитного скоринга на основе алгоритмов дерева решений и провести анализ качества полученной модели.

### Основные задачи:
- изучить предметную область кредитного скоринга;
- провести анализ и предобработку данных;
- обучить и сравнить модели Decision Tree и Random Forest;
- оценить качество моделей с использованием стандартных метрик;
- реализовать веб-интерфейс для работы с моделью;
- обеспечить наглядную интерпретацию результатов.

## Данные

В проекте используется датасет **German Credit Data**.

### Характеристики датасета:
- количество наблюдений: 1000
- количество признаков: 20
- целевая переменная: `kredit`
  - 1 — заёмщик с высоким кредитным риском
  - 0 — надёжный заёмщик

Признаки представлены числовыми значениями, включая закодированные категориальные признаки.

---

## Предобработка данных

В ходе подготовки данных были выполнены следующие шаги:
- загрузка и первичный анализ данных;
- разделение данных на обучающую и тестовую выборки;
- формирование pipeline для обучения моделей;
- сохранение списка используемых признаков.

Дополнительная нормализация данных не требовалась, так как используемые модели основаны на деревьях решений.

---

## Модели машинного обучения

В рамках проекта были реализованы и сравнены следующие модели:

- **Decision Tree Classifier**
- **Random Forest Classifier**

Для оценки качества моделей использовались следующие метрики:
- ROC-AUC
- F1-score
- Recall
- Precision

По результатам сравнения в качестве финальной модели был выбран **Random Forest**, показавший наилучшее качество.

### Итоговые метрики финальной модели:
- ROC-AUC: 0.8361
- F1-score: 0.8684
- Recall: 0.9429
- Precision: 0.8049

## Веб-приложение

Проект реализован в виде Flask-приложения и включает две основные страницы:

### 1. Страница скоринга
- ввод параметров заёмщика;
- подсказки и описания признаков;
- отображение вероятности риска;
- автоматическое принятие решения (одобрение / отказ);
- визуальное выделение наиболее значимых признаков.

### 2. Страница информации о модели
- основные метрики качества модели;
- ROC-кривые;
- матрицы ошибок;
- список наиболее значимых признаков и их важность.
